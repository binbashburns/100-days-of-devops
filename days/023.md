# Day 023: Fork a Git Repository

**Time**: 3 minutes

## Objective
Fork a git repository under another user

## Technologies Used

## Steps

1. Log into gitea, navigate to desired repository

2. Click **Fork** on the repo

    ![gitea2](../resources/gitea2.png)
