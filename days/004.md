# Day 004: Script Execution Permissions

**Time**: 4 minutes

## Objective
Your task is to grant executable permissions to the `/tmp/xfusioncorp.sh` script on `App Server 2`. Additionally, ensure that all users have the capability to execute it.

## Technologies Used
- Users and groups in Linux, `chmod`

## Steps

1. SSH to `stapp02`

    ```bash
    thor@jumphost ~$ ssh steve@stapp02
    [steve@stapp02 ~]$
    ```

2. Make the script RWX for Owner (7), RX for Group, RX for Others

    ```bash
    # Modify permissions to 755
    [steve@stapp02 ~]$ sudo chmod 755 /tmp/xfusioncorp.sh 

    # Verify changes:
    [steve@stapp02 ~]$ ls -la /tmp/xfusioncorp.sh 
    -rwxr-xr-x 1 root root 40 Dec  2 01:54 /tmp/xfusioncorp.sh
    [steve@stapp02 ~]$
    ```

## Key Takeaways
- Never forget Linux [user and group permissions schemas](https://www.redhat.com/en/blog/manage-permissions)
