# Day 036: Deploy Nginx Container on Application Server

**Time**: 5 minutes

## Objective
The Nautilus DevOps team is conducting application deployment tests on selected application servers. They require a nginx container deployment on `Application Server 1`. Complete the task with the following instructions:
- On `Application Server 1` create a container named `nginx_1` using the `nginx` image with the `alpine` tag. Ensure container is in a running state.

## Technologies Used
- `ssh`
- `docker`

## Steps

1. SSH into `stapp01` server

    ```bash
    # SSH to stapp01
    thor@jumphost ~$ ssh tony@stapp01
    ```

2. Spin up docker container

    ```bash
    # Run nginx:alpine and call it nginx_1
    [tony@stapp01 ~]$ sudo docker run -d --name nginx_1 -p 80:80 nginx:alpine
    ```

3. Verify it's running

    ```bash
    # Check running containers
    [tony@stapp01 ~]$ sudo docker ps -a
    # CONTAINER ID   IMAGE          COMMAND                  CREATED          STATUS          PORTS                NAMES
    # 9da9b76cde75   nginx:alpine   "/docker-entrypoint.â€¦"   13 seconds ago   Up 11 seconds   0.0.0.0:80->80/tcp   nginx_1
    ```
