# Day 035: Install Docker Packages and Start Docker Service

**Time**: 5 minutes

## Objective
The Nautilus DevOps team aims to containerize various applications following a recent meeting with the application development team. They intend to conduct testing with the following steps:

- Install `docker-ce` and `docker compose` packages on `App Server 3`.

- Initiate the `docker` service.

## Steps

1. SSH into `stapp03` server, research docker official instructions

    ```bash
    # SSH to stapp03
    thor@jumphost ~$ ssh banner@stapp03

    # Find docker instructions for RHEL-type instances: https://docs.docker.com/engine/install/rhel/
    ```

2. Set up the docker repository, install docker packages, enable docker

    ```bash
    # Set up the docker repository
    [banner@stapp03 ~]$ sudo dnf -y install dnf-plugins-core
    [banner@stapp03 ~]$ sudo dnf config-manager --add-repo https://download.docker.com/linux/rhel/docker-ce.repo

    # Install the docker packages
    [banner@stapp03 ~]$ sudo dnf install docker-ce docker-ce-cli containerd.io docker-buildx-plugin -y

    # Enable Docker
    [banner@stapp03 ~]$ sudo systemctl enable --now docker
    ```
